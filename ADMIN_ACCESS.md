# Как попасть в админ-панель

## Шаг 1: Узнайте ваш Telegram User ID

1. Откройте бота [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте команду `/start`
3. Бот покажет ваш User ID (например: `123456789`)

## Шаг 2: Установите OWNER_ID в боте

1. Перейдите в настройки вашего бота на Railway/Render/другом хостинге
2. Добавьте переменную окружения:
   - **Имя**: `OWNER_ID`
   - **Значение**: ваш Telegram User ID (только цифры, без кавычек)
3. Перезапустите бота

## Шаг 3: Откройте Mini App

1. Откройте Mini App через бота
2. В нижней навигации появится кнопка **"Admin"** (⚙️)
3. Нажмите на неё, чтобы открыть админ-панель

## Альтернативный способ (для тестирования)

Если кнопка не появляется, вы можете открыть админ-панель напрямую, добавив параметр `?admin=true` к URL Mini App.

## Отладка

Если админ-панель не работает:

1. Откройте консоль браузера (F12 → Console)
2. Проверьте сообщения об ошибках
3. Убедитесь, что:
   - `OWNER_ID` установлен в переменных окружения бота
   - Значение `OWNER_ID` совпадает с вашим Telegram User ID
   - Бот перезапущен после добавления переменной окружения
   - API бота доступен и отвечает

## Что вы увидите в админ-панели:

- **Статистика**: общее количество пользователей, онлайн, активных за 24ч, отправлено яиц, вылуплено яиц, всего поинтов
- **Управление Tasks**: список всех задач, возможность добавлять новые и удалять существующие
